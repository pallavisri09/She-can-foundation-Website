# She Can Foundation Website 🌸

A responsive and visually appealing website created for **She Can Foundation** using **HTML** and **CSS**.
This project is designed to spread awareness about women empowerment, encourage volunteering, and provide a clean and modern user experience.

---

## 📌 Project Overview

The **She Can Foundation Website** is a simple front-end web project developed to showcase the mission and vision of a women empowerment organization.
The website includes different sections such as:

* Home Section
* About Section
* Mission & Vision
* Volunteer / Join Us Button
* Footer Section

The design focuses on:

* Clean layout
* Attractive color combinations
* Responsive design
* User-friendly interface

---

## 🚀 Features

✨ Modern and professional UI design
